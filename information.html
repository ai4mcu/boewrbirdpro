📋 프로그램 소개

BOWERBIRDPRO는 시리얼 또는 블루투스 LE(BLE) 장치에 연결하여 데이터를 수신하고, 실시간으로 그래프에 표시하며, CSV 파일로 저장할 수 있는 프로그램입니다.

🚀 사용법

연결 설정

  • 언어: 드롭다운에서 원하는 표시 언어를 선택하세요.
  • 장치 유형: 유선 연결은 '시리얼 포트', BLE 장치는 '블루투스 LE'를 선택하세요.
  • 시리얼 포트: 장치가 연결된 COM 포트와 보드레이트를 선택하세요.
  • 블루투스 LE: 'BLE 장치 스캔'을 클릭하고 장치를 선택한 후 '연결'을 클릭하세요.

센서 구성

  • 센서 개수: 장치가 보낼 데이터 값의 개수를 지정하세요 (최대 10개).
  • 센서 이름: 각 센서에 대해 사용자 정의 이름을 설정할 수 있습니다.

데이터 형식 설정

연결된 장치는 다음 형식으로 데이터를 전송해야 합니다:

DATA,값1,값2,값3,...

예시 (3개 센서):

DATA,25.5,1023.1,75.0

주의: 'DATA,' 다음의 값 개수가 설정한 센서 개수와 일치해야 합니다.

로깅 및 플로팅

  • 로깅 시작: 데이터 수신, 그래프 표시, CSV 저장을 시작합니다.
  • CSV 파일: 프로그램과 같은 폴더에 연결 정보와 타임스탬프가 포함된 파일명으로 자동 생성됩니다.
  • 일시정지/재개: 데이터 로깅을 일시적으로 중지하거나 계속할 수 있습니다.
  • 로깅 중지: 현재 세션을 종료하고 CSV 파일을 마무리합니다.

실시간 그래프

  • X축: 시간 (시:분:초 형식)
  • Y축: 센서 값 (자동 범위 조정)
  • 각 센서는 고유한 색상의 선으로 표시됩니다
  • 범례로 각 센서를 구분할 수 있습니다

⚙️ 기술적 특징

  • 비동기 통신: 시리얼 및 BLE 통신을 위한 별도 워커 스레드
  • 실시간 처리: UI 블로킹 없는 데이터 수신 및 처리
  • 자동 프로토콜 감지: Nordic UART, JDY_FFE0 등 다중 BLE 프로파일 지원
  • 데이터 검증: 실시간 데이터 유효성 검사 및 오류 처리
  • 메모리 관리: 최대 300개 데이터 포인트 유지 (rolling window)

🔧 호환 장치

지원 MCU: Arduino, Micro:bit, Raspberry Pi Pico, ESP32 등 모든 MCU

통신 방식:

  • 시리얼 통신: USB, UART, SPP 블루투스
  • BLE 통신: Micro:bit, ESP32, Arduino Nano 33 BLE 등

주의사항: 블루투스 전송 시 센서가 많으면 데이터가 분할 전송될 수 있으므로, 많은 센서 사용 시 시리얼 통신 권장
